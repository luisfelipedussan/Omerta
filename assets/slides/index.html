Tradicional self-healing systems:  automatically detect/diagnose, and recover from failures

But why settle for a rigid system when a flexible, self-healing one can adapt and autonomously heal using atomic actions and dynamic predicates?

We propose a novel self-healing framework that learns recovery strategies for fine grained system healing behavior at run time and changes predicates dynamicaly.

The framework has three main components 

Fault detection

Learning model

  • The monitor uses the information given by the fired predicate to detect the atomic action that trigger the activation.
  • The monitor uses a Q-Learning agent to establish the reward for the action taken at the current state based on the predicated current value.
  • The learning processes maps states to high value atomic actions
